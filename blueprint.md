# FindMySens - FPS Mouse Sensitivity Checker

## 프로젝트 개요
FPS 게임 유저들이 자신의 마우스 감도가 적절한지 확인하고 연습할 수 있는 웹 기반 도구입니다.

## 주요 기능
1. **중앙 집중형 테스트 구역**: 화면 중앙의 제한된 영역에서 감도 테스트 진행. (16:9 비율)
2. **그리드 타겟 시스템**: 직사각형 영역 내에서 정사각형 모양의 타겟들로 구성된 그리드.
3. **인터랙티브 테스트**: 점차 빨라지는 속도로 무작위 칸이 점등되며, 사용자가 이를 클릭하여 정확도와 반응속도를 측정. (1.0s -> 0.3s)
4. **테스트 제어**: 'START'로 시작하고, 테스트 도중 'STOP' 버튼으로 즉시 중단 가능.
5. **결과 피드백**: 명중률(Hits), 미스(Misses), 그리고 현재 점멸 속도 정보 제공.

## 디자인 가이드라인 (Modern Baseline)
- **색상**: 다크 배경 (#0f172a) + 네온 그린/블루 하이라이트.
- **레이아웃**: 16:9 비율의 중앙 그리드 컨테이너.
- **애니메이션**: 타겟 활성화 시 부드러운 글로우(Glow) 효과.
