# FindMySens - FPS Mouse Sensitivity Checker

## 프로젝트 개요
FPS 게임 유저들이 자신의 마우스 감도가 적절한지 확인하고 연습할 수 있는 웹 기반 도구입니다.

## 주요 기능
1. **중앙 집중형 테스트 구역**: 화면 중앙의 제한된 영역에서 감도 테스트 진행.
2. **그리드 타겟 시스템**: 작은 사각형들로 쪼개진 그리드 내에서 타겟이 생성됨.
3. **인터랙티브 테스트**: 1초 간격으로 무작위 칸이 점등되며, 사용자가 이를 클릭하여 정확도와 반응속도를 측정.
4. **결과 피드백**: 명중률(Hits)과 미스(Misses) 통계 제공.

## 디자인 가이드라인 (Modern Baseline)
- **색상**: 다크 배경 (#0f172a) + 네온 그린/블루 하이라이트.
- **레이아웃**: Flexbox와 Grid를 사용한 완전한 중앙 정렬.
- **애니메이션**: 타겟 활성화 시 부드러운 글로우(Glow) 효과.
