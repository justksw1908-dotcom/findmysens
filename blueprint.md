# FindMySens - FPS Mouse Sensitivity Checker

## 프로젝트 개요
FPS 게임 유저들이 자신의 마우스 감도가 적절한지 확인하고 연습할 수 있는 웹 기반 도구입니다.

## 주요 기능
1. **중앙 집중형 테스트 구역**: 화면 중앙의 제한된 영역에서 감도 테스트 진행. (16:9 비율 유지)
2. **그리드 타겟 시스템**: 
    - **Standard**: 16x9 그리드.
    - **Small**: 32x18 그리드 (타겟 크기가 절반으로 감소).
3. **인터랙티브 테스트**: 점차 빨라지는 속도로 타겟 점등. (1.0s -> 0.3s)
4. **테스트 종료 조건**: 5번의 미스 발생 시 테스트 자동 종료.
5. **감도 분석 엔진**:
    - 이전 타겟 대비 다음 타겟으로 이동 시의 클릭 오차 분석.
    - 오버슈트(Overshoot)와 언더슈트(Undershoot) 데이터를 수집.
    - 테스트 종료 후 감도 조정(UP/DOWN) 가이드 제공.
6. **테스트 제어**: 'START', 'STOP', 'Grid Size Selection' 기능.

## 디자인 가이드라인 (Modern Baseline)
- **색상**: 다크 배경 + 네온 테마.
- **레이아웃**: 1120x630 대형 그리드 컨테이너.
- **결과 화면**: 시각화된 분석 데이터와 감도 추천 텍스트.
